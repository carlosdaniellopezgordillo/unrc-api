# Frontend React para API Vinculación UNRC

Este frontend permite a los usuarios interactuar visualmente con la API: registro, login y visualización de datos principales.

## Instalación y ejecución

1. Entra a la carpeta `frontend`:
   ```bash
   cd frontend
   ```
2. Instala las dependencias:
   ```bash
   npm install
   ```
3. Inicia la aplicación:
   ```bash
   npm start
   ```

La app estará disponible en http://localhost:3000

## Características
- Registro y login de usuarios
- Visualización de estudiantes, empresas y oportunidades
- Diseño moderno y responsivo (color principal #9F2241)

---

> Recuerda que la API debe estar corriendo en http://localhost:8000 para que el frontend funcione correctamente.
